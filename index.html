このサイトは、英検5級レベルの基礎から2級レベルの読解・英作文発想までを、 視覚的な解説モードと、1問ずつ反復できる演習モードで学べるようにした静的学習サイトです。 Unityアプリの設計感覚に寄せて、「主語はゲームオブジェクト」「動詞は Update 関数の中心処理」のように説明します。

すぐ1問ずつ練習する 解説モードから始める
日本語の感覚: 気持ちや場面から後ろで確定しやすい
きのう わたしは 公園で 遊んだ
話し手の視点で、情報をあとから積み重ねても通じやすい。
英語の感覚: 主語 + 動詞を先に固定して全体を俯瞰する
I played in the park yesterday
先に文の骨組みを置くので、英文は「設計図」っぽく見える。

このサイトでできること

基礎→実戦までを飛ばさず、でも退屈にしないために、解説・比較・演習・統計を1つにまとめています。

1. 級別ロードマップ

5級の be 動詞から、2級レベルの長文・意見表明まで段階的に整理しています。

2. 解説モード

日本語と英語の語順や視点の違いを、図・色・比喩つきで楽しく理解できます。

3. 反復演習モード

3回は答えや解説を見ながら練習し、4回目で何も見ずに解く設計です。

4. 実戦モード

最初から完全に見ずに解き、1問ごとに詳しいポップアップ解説で復習できます。

級別ロードマップ

英検の各級目安を参考に、「どこまで覚えると次へ進めるか」を見える化しました。語彙・文法・読解の伸び方も並べています。

基礎の土台 5級: 英語の最初の設計図を覚える

be動詞 / 一般動詞 / 疑問詞 / can / 現在進行形。短い文を正しい順番で置く力を作る段階。

  • 主語 + 動詞の骨組みを毎回意識する
  • 家族・学校・食べ物など身近な語彙を固める
  • Yes/No疑問文と WH疑問文に慣れる
時制が広がる 4級: 過去・未来・比較で話を広げる

過去形 / 未来表現 / 比較級 / 不定詞 / 接続詞。理由や予定を言えるようにする段階。

  • いつの話かを時制で固定する
  • because / when / if で文をつなぐ
  • 予定・希望・比較を自然に言う
中学完成 3級: 説明する英語に進化する

現在完了 / 受動態 / 関係代名詞 / 間接疑問 / 短い英作文。理由を添えて答える段階。

  • 情報を1文でまとめる感覚を育てる
  • だれが何をされたかを受動態で捉える
  • 25〜35語で自分の考えを書く準備をする
高校中級 準2級: 論理のつながりを追う

関係副詞 / 仮定法の入口 / 長めの読解 / まとまった英作文。言いたいことを段落で整理する段階。

  • 文どうしの関係を接続表現で読む
  • 50語前後で理由を2つ並べて書く
  • 日常から社会寄りテーマへ広げる
高校卒業 2級: 社会的テーマを俯瞰して語る

抽象語彙 / 論説読解 / 要点把握 / 80〜100語の意見英作文。根拠を組み立てて伝える段階。

  • 環境・教育・技術などの文章に慣れる
  • 賛成/反対と理由を論理的に配置する
  • 長い1文でも主語・動詞を見失わない

解説モード: 日本語の主観と、英語の俯瞰構造を楽しく理解する

英語が苦しくなる原因は、「単語を知らない」だけではなく、文の組み立て方の前提が日本語と違うことにあります。ここを図で掴むと、問題演習が一気に楽になります。

構造の違い 英語は「カメラを先に固定」する言語

英語はまず誰の話か（主語）と何をする/どうであるか（動詞）を決めてから、場所・時間・理由を足します。

  • Unityでいうと、まず GameObject と main script を置く
  • そのあとに position / effect / log を足すイメージ
  • だから文の前半が崩れると全部崩れやすい
be動詞と一般動詞 「状態」か「動作」かでエンジンが変わる

I am happy. は状態、I play tennis. は動作。英語は1文1エンジンが基本なので、be動詞と一般動詞を同時に主役にしません。

  • 状態 = Inspector に表示される現在値
  • 動作 = 実際に走るメソッド
  • 主語に合わせた形変化を毎回確認する
時制の感覚 時制は「いつのセーブデータを見るか」

現在形はふだん設定、過去形は過去ログ、現在完了は「今に影響が残る履歴」。ただの和訳ではなく時間軸で覚えると崩れません。

  • played = 昨日終わったイベント
  • have played = 履歴が今の状態にもつながる
  • will play = 未来イベントの予定
日本語の並び
きのう わたしは 図書館で 勉強した
英語の並び
I studied in the library yesterday

英語は 主語 + 動詞 を先に確定してから後ろへ拡張するので、「まず何が起きる文か」を先に決めるのがコツです。

演習モード: 1問ずつ、見ながら育てる / 見ずに試す

級・出題順・学習モードを切り替えながら進めます。各問題には、答え・理由・日本語との違い・プログラミング比喩つきの詳しい解説があります。

反復モード（学習モード）

同じ問題は 1〜3回目まで解説を自由に見てOK。4回目だけ完全に見ずに解きます。理解→定着の流れを1問単位で回します。
