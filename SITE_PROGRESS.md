# サイト進捗管理 — 日英文法ビジュアル比較

## 計画

日本語と英語の文法構造の違いを視覚的に比較しながら学べるインタラクティブ学習サイト。

### 設計方針

- 単一 HTML ファイル（外部依存なし）
- GitHub Pages 静的サイト対応
- ライトモードのみ・モバイルレスポンシブ
- 色分け・図解・アニメーションによる直感的な理解
