# 英検5級→2級 学習サイト 進捗計画

## 目的
- 英検5級レベルから英検2級レベルまでを、静的サイト1本で段階的に学べるようにする。
- 日本語と英語の構造差を視覚的に説明する「解説モード」と、反復演習できる「演習モード」を両立する。
- Unity/プログラミングの発想に置き換えた説明を混ぜ、エンジニアにも刺さる理解を目指す。

## 設計方針
- ルート直下に `index.html` を置く単一ページ構成にして、GitHub Pages 上でそのまま動くようにする。
- 外部ライブラリは追加せず、HTML/CSS/JavaScript だけで完結させる。
- 問題データはサイト内に内包し、級別・分野別に整理する。
- 解説はポップアップで開き、ページ遷移なしで詳細確認できるようにする。
- 学習履歴は `localStorage` へ保存し、各問題の挑戦回数と正答数を保持する。

## 学習サイトの主要機能
- [ ] 級別ロードマップ（5級 / 4級 / 3級 / 準2級 / 2級）
- [ ] 日本語と英語の語順比較図、文法の俯瞰図、プログラミング比喩つき解説
- [ ] 演習モードA: 1〜3回目は答え/解説を自由に見ながら、4回目は見ずに解く反復モード
- [ ] 演習モードB: 最初から完全に見ずに解く実戦モード
- [ ] 出題順切り替え（ランダム / 全問を順番に網羅）
- [ ] ポップアップで詳しい解説、和訳比較、プログラム比喩を表示
- [ ] 学習統計（挑戦数、正答率、現在の周回状況）

## データ準備方針
- [ ] 英検の級目安に沿って、5級〜2級の典型文法・語彙・読解テーマを整理する
- [ ] 公開サイトには転載過去問ではなく、級別傾向に沿ったオリジナル練習問題を多数収録する
- [ ] 各問題に「答え」「なぜそうなるか」「日本語との違い」「プログラミング比喩」を付ける
